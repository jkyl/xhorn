February 23, 2016 - ADC Testing

Here is a class for sampling on the ROACH's ADC. It imports fit_snap from fit_cores but nothing from rww_tools because that code is a nightmare. Includes method for fitting and clearing ogp. Will update with fft method. Use like this to fit ogp and look at a sample:

r = Roach(IP) r.clear_ogp() r.fit_ogp() r.plot_snap() r.plot_cores()
