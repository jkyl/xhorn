Notes on New Oscillator and Mixer

For future reference, some specs for the Herley CTI 9506 MHz oscillator and the Mini Circuits wide band frequency mixer:

Oscillator:

  • +13 dBm output power
  • 0 dBm REF CLK input power
  • 95.1 MHz, 47.5 MHz, or 23.8 MHz REF CLK input freq.
  • 15V DC supply

Mixer

  • 3200-15000 MHz LO/RF input freq.
  • DC-4000 MHz IF output freq.
  • 6.6 dB typ conversion loss
  • 15V DC supply

Preliminary electronics setup. Eventually the spectrum analyzer will be replaced with the roach board, at which point we'll need a new reference clock. Note the 20dB attenuator on the input of the analyzer, and the 20dB amplifier between the 50 MHz ref. output of the analyzer and the ref. input of the oscillator.

Mixer functions as expected: with 9.5 GHz RF and 9.4 GHz LO inputs, we see the IF output peak at exactly 100 MHz on the analyzer. The 18.9 GHz summed-frequency component is also present, but only ~15 dB above noise, compared to ~30dB for the differenced-frequency component (lowpassed)? We also see both the RF and LO peaks.
